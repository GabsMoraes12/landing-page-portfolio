# Landing Page Responsiva | Gabriel Moraes

Este projeto demonstra a aplicação prática de desenvolvimento web front-end combinado com conceitos estratégicos de Marketing Digital e Design de Interface (UI/UX). Trata-se de uma página de captura estruturada do zero, focada em alta conversão e adaptabilidade para diferentes dispositivos (computador e celular).

---

## Tecnologias e Conceitos Utilizados

* **HTML5:** Estruturação semântica da página de captura.
* **CSS3:** Estilização visual completa, incluindo layout responsivo, tipografia corporativa e design baseado em componentes.
* **UI/UX Design:** Aplicação de conceitos de experiência do usuário, hierarquia visual e contraste de cores para direcionar a atenção do visitante.
* **Copywriting & Marketing:** Conteúdo focado em destacar competências de um perfil profissional híbrido com foco em conversão/conexão profissional.

---

## Funcionalidades da Página

1. **Design Responsivo:** A página adapta-se perfeitamente a telas de desktops, tablets e smartphones, garantindo uma boa experiência de navegação em qualquer dispositivo.
2. **Identidade Visual Limpa:** Uso de uma paleta de cores moderna e balanceada para transmitir profissionalismo e clareza.
3. **Chamada para Ação (CTA):** Botão de conversão destacado de forma estratégica para incentivar o visitante a se conectar diretamente no LinkedIn.
4. **Seção de Habilidades (Tags):** Exibição dinâmica das principais competências operacionais (Excel, Design, Redes Sociais e Lógica).
